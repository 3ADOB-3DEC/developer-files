Developer Files (Разработчик файлов) - программа для быстрого создания неограниченного количества различного типа файлов.

Данная программа ограничена:

  1. Список разрешенных файлов ограничен. В него входят файлы c расширением html, css, js, php.
  2. Определенный синтаксис для работы. Для создания файла нужна определённая команда. Для указания параметров нужно указать атрибут.
  3. Программа создаёт файлы определенного формата за раз.

Данная программа позволяет:

  1. Не определять все параметры.
  2. Множественное создание файлов за раз.
  3. Работать можно не запуская программу каждый раз при создании файла.

Список команд: @html, @css, @js,@ php;

Список атрибутов: s=, f=;

Синтаксис

  • @css s=C://project/css/stylemain f=style-1,style-2
    В этом случае программа создаст два файла style-1.css и style-2.css по указанному пути C://project/css/stylemain.
  • @php f=index,main
    В этом случаии программа создаст в текущем каталоги исполнения программы два файла index.php и main.php
  • @html s=./pages
    В этом случае программа создаст каталог в текущем каталоги исполнения программы и в нём файл file.html
  • @js
    В этом случае программа создаст в текущем каталоги исполнения программы файл file.js

Атрибуты:

  • s=[путь]. Позволяет указывать путь к файлом. Принимает два вида путей к файлом. Относительный (./) текущий каталог. Абсолютный, указывает полный путь от корня корня файловой системы (локального диска) до его расположения.

    С://pojects/pages/main/version/one - Абсолютный
    ./version/one - Относительный

  • f=[имя(имена)] Позволяет указывать имя(имена) файла(ов). Синтаксис

    f=index или f=index,main,stage и т.д.
